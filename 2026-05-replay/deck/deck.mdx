# An agent

Structured, type-safe output with Pydantic AI.

```py
from datetime import date, datetime
from pydantic import BaseModel
from pydantic_ai import Agent

class ConferenceTalk(BaseModel):
    title: str
    start_time: datetime
    room: str | None = None

conference_talk_agent = Agent(
    "gateway/anthropic:claude-haiku-4-5",
    output_type=list[ConferenceTalk],
    instructions="Extract information about upcoming meetings",
)
```

# Make the agent durable

One line (and an import) to make the agent durable.

```py
...

from pydantic_ai.durable_exec.temporal import TemporalAgent

durable_conference_talk_agent = TemporalAgent(conference_talk_agent)
```

**No need to manually wrap your tool calls as activities.**

</Slide>

<Slide tab="durable">

# How this works

<div style={{ display: 'grid', gridTemplateColumns: '1.1fr 1fr', gap: '2rem', alignItems: 'start' }}>

<div>

- The **agent loop** lives in the workflow - completely deterministic
- **Every external call** becomes an activity:
  - model requests
  - `@agent.tool` functions
  - MCP server calls
- One `TemporalAgent` wraps your agent and wires it all up
- If the worker dies mid-run, Temporal replays the workflow and resumes - without re-paying for the model calls already done.
